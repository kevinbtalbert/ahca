Data Book - SMMC Re-Procurement











UPDATED DATA BOOKThe Agency for Health Care Administration (Agency) has posted an updated data book providing relevant background information that prospective plans will find useful in the development of their response to the Invitation to Negotiate (ITN).  The data book consists of a comprehensive set of utilization and spending data consistent with actuarial rate-setting practices and standards.  The data book is contained in a ZIP file which includes multiple PDFs and spreadsheets pertaining to both the Managed Medical Assistance (MMA) and Long-term Care (LTC) programs.Updated Data Book published April 4, 2023
