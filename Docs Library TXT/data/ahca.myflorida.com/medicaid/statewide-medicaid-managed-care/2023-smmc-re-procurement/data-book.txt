Data Book











The Agency for Health Care Administration (AHCA or Agency) has posted a data book providing relevant background information that prospective plans will find useful in the development of their response to the Invitation to Negotiate (ITN).  The data book consists of a comprehensive set of utilization and spending data consistent with actuarial rate-setting practices and standards.  The data book is contained in a ZIP file which includes multiple PDFs and spreadsheets pertaining to both the Managed Medical Assistance (MMA) and Long-term Care (LTC) programs.
